# UI sound effects

Both CC0 (Public Domain) from [freesound.org](https://freesound.org), by
[Breviceps](https://freesound.org/people/Breviceps/) ("Clicks, Buttons & UI
sounds" pack). No attribution required, credited here anyway.

- `click.mp3` - default button click, every interactive element.
  [Wet Click](https://freesound.org/people/Breviceps/sounds/448080/)
- `close.mp3` - close/back buttons only, played a touch pitched-down for a
  "settling down" feel distinct from the open click.
  [Plop!](https://freesound.org/people/Breviceps/sounds/447910/)
